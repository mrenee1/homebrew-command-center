# Command Center — Homebrew Tap

A little Mac app to install, browse, and copy your Claude Code skills.
Drop a `SKILL.md`, a `README.md`, a skill folder, or a `.zip` onto it and it lands in
`~/.claude/skills/` automatically.

## Install

```sh
brew tap USERNAME/command-center
brew install --cask command-center
```

Then open **Command Center** from your Applications folder.

> First launch: because the app isn't notarized by Apple, right-click the app and
> choose **Open** the first time, then confirm.

## Update

```sh
brew upgrade --cask command-center
```

## Uninstall

```sh
brew uninstall --cask command-center
```
